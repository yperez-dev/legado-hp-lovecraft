# El legado de H.P. Lovecraft

Sitio web sobre la biografía de Howart Phillips Lovecraft, su vida, su vida, legado y la influencia que a tenido a lo largo de los años en el horror cósmico.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript vanilla

## Características del sitio web

* Barra de navegación interactiva generada con JavaScript
